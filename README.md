# kilo-editor
Um editor de texto simples construído em C

Este é um editor simples escrito puramente em C que usei para entender melhor sobre terminais, programação em baixo nível e editores de texto.
Segui todos os passos descritos em https://viewsourcecode.org/snaptoken/kilo/index.html
Escrevi todo o código manualmente para absorver todos os detalhes – por mais que minha deficiência em conhecimentos em C e programação em baixo nível não tenham permitido entender 100% do código.
